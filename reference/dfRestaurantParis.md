# Parisian restaurants

Parisian restaurants (Restaurants parisiens)

## Usage

``` r
dfRestaurantParis
```

## Format

### `dfRestaurantParis`

A data frame with 13,823 rows and 17 columns:

- x:

  longitude

- y:

  latitude

- french:

  French restaurant

- asian:

  asian restaurant

- caribbean:

  caribbean restaurant

- maghreb:

  maghreb restaurant

- african:

  african restaurant

- european:

  european restaurant

- south_america:

  south_american restaurant

- indian:

  indian restaurant

- world:

  world food restaurant

- cafeteria:

  cafeteria restaurant

- tea_house:

  tea_house

- brasserie:

  brasserie restaurant

- fastfood:

  fastfood restaurant

- bar:

  bar

- restaurant:

  restaurant

## Source

source : https://opendata.paris.fr
